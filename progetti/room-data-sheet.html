Room Data Sheet
Automatizzati da Revit

Generazione automatica di schede ambiente in formato HTML e PDF direttamente da un modello Revit, usando uno script Python all'interno di Dynamo.

→ Inserisci qui screenshot o timelapse del workflow

Il problema

In un progetto complesso, le schede ambiente devono essere generate e aggiornate manualmente ad ogni revisione del modello. Con decine o centinaia di ambienti, questo processo è lento e soggetto a errori.

La soluzione

Uno script Python in Dynamo legge i parametri di ogni ambiente (nome, numero, area, altezza soffitto, finiture) e genera dinamicamente un file HTML per ogni stanza. WeasyPrint converte poi i file in PDF.

  • Lettura parametri via LookupParameter e StorageType
  • Conversione unità interne Revit → cm
  • Template HTML con CSS inline per layout stabile
  • Gestione errori silenziosi e parametri vuoti

Snippet principale

# Recupero altezza soffitto dall'elemento ceiling
def get_ceiling_height(ceiling, doc):
    bb = ceiling.get_BoundingBox(None)
    if bb is None:
        return None
    mid_pt = XYZ(
        (bb.Min.X + bb.Max.X) / 2,
        (bb.Min.Y + bb.Max.Y) / 2,
        bb.Min.Z + 0.1  # offset Z sopra il pavimento
    )
    room = doc.GetRoomAtPoint(mid_pt)
    if room:
        h = room.LookupParameter("Unbound Height")
        if h and h.HasValue:
            return h.AsDouble() * 30.48  # ft → cm
    return None
